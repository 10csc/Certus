- **双入口架构**：`main.cpp` 实现无头 CLI 后端（支持 JSON/人类可读输出），`main_gui.cpp` 启动 Qt 桌面应用并集成暗色主题与单实例保护。
- **核心管理层**：`AgentManager` 负责 Python 子进程的生命周期管理（握手、心跳、崩溃检测），`BrowserManager` 控制 CDP 浏览器实例，`Database` 封装 SQLite 操作（WAL 模式）。
- **UI 模块化设计**：`MainWindow` 采用侧边栏导航配合 `QStackedWidget` 路由，将功能拆分为独立的页面组件（如 `SearchPage`, `ConfigPage`）。
- **工具支撑层**：`utils` 目录提供跨平台加密（Windows DPAPI）、滚动日志系统及 JSON 解析工具，确保核心逻辑的健壮性与安全性。
