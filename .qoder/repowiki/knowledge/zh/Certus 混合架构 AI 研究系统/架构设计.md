- **跨语言进程编排**：C++ `AgentManager` 作为核心控制器，通过 `QProcess` 启动并监控 Python 代理脚本（`agent.py`），建立基于 JSON 帧的 `stdin/stdout` 通信信道。
- **生命周期同步**：实施严格的握手协议（`hello/hello_ack`）与心跳检测机制（30s ping/35s timeout），确保宿主应用能实时感知代理状态并在崩溃时触发进化逻辑。
- **统一数据与配置边界**：所有子模块共享本地 SQLite 数据库（`certus.db`）进行会话状态管理与加密配置存储，C++ 层负责浏览器实例（CDP）的生命周期管理，Python 层专注于业务逻辑执行。
